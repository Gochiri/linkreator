# LinKreator

## Description
LinKreator es un sistema de creación de contenido para LinkedIn que genera posts, carruseles e imágenes listas para publicar, con un tono de marca definido (estilo Dan Koe / Justin Welsh) dirigido a una audiencia interesada en automatizaciones, IA y soluciones innovadoras. Transforma ideas en contenido de alto impacto con hashtags optimizados.

## Problems & Solutions

### Problem 1: Baja frecuencia de publicación
LinKreator elimina el bloqueo creativo generando contenido variado y listo para publicar, permitiendo mantener una presencia constante en LinkedIn sin fricción.

### Problem 2: Falta de consistencia en el tono de marca
El sistema tiene pre-configurado un tono directo, minimalista y accionable (estilo Dan Koe / Justin Welsh) que se aplica automáticamente a todo el contenido generado.

### Problem 3: Crear carruseles e imágenes toma demasiado tiempo
Genera automáticamente diseños visuales (carruseles, imágenes de apoyo) alineados con la marca personal, listos para descargar y publicar.

### Problem 4: No saber qué hashtags usar
Incluye hashtags optimizados y relevantes al nicho (IA, automatización, innovación) en cada publicación generada.

### Problem 5: Avatar de audiencia difuso
El avatar está pre-definido: profesionales y emprendedores que buscan automatizar, innovar y resolver problemas con tecnología. Todo el contenido se dirige a esta audiencia.

## Key Features
- Generador de posts con tono de marca (corto, directo, storytelling + frameworks)
- Creador de carruseles con diseño visual profesional
- Generador de imágenes para posts
- Hashtags optimizados por nicho automáticos
- Avatar/audiencia pre-definida y configurable
- Variedad de formatos: tips, storytelling, casos de estudio, opiniones
- Configuración de tono de marca personalizable
- Vista previa de contenido antes de publicar
- Calendario de contenido para planificación
