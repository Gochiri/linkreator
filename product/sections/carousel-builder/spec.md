# Carousel Builder

## Overview
Editor visual para crear carruseles de LinkedIn slide por slide. Permite diseñar cada diapositiva con texto, fondo y elementos visuales, manteniendo consistencia con la marca personal. Incluye plantillas pre-diseñadas y vista previa del carrusel completo.

## User Flows
- Elegir una plantilla de carrusel o empezar desde cero
- Agregar, eliminar y reordenar slides
- Editar texto, fondo y layout de cada slide
- Previsualizar el carrusel completo con navegación slide por slide
- Exportar como PDF o imágenes individuales

## UI Requirements
- Panel de slides en miniatura (sidebar izquierdo) con drag & drop
- Canvas central de edición del slide activo
- Toolbar superior con opciones de texto, color, layout
- Galería de plantillas de carrusel pre-diseñadas
- Vista previa fullscreen del carrusel con navegación
- Contador de slides (máximo 20 para LinkedIn)
- Diseño estilo editor de presentaciones pero minimalista
