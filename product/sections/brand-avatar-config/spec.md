# Brand & Avatar Config

## Overview
Centro de configuración de la marca personal y el avatar de audiencia. Permite definir el tono de voz (inspirado en Dan Koe / Justin Welsh), configurar el perfil del creador, definir el avatar ideal de audiencia, y gestionar la biblioteca de hashtags por categoría.

## User Flows
- Configurar perfil del creador (nombre, título, industria, bio)
- Definir tono de marca con ejemplos y referencias
- Crear y editar el avatar de audiencia ideal (demographics, pain points, goals)
- Gestionar biblioteca de hashtags organizados por categoría
- Previsualizar cómo se aplica el tono en un post de ejemplo

## UI Requirements
- Formulario de perfil del creador con foto y datos
- Panel de tono de marca con sliders (formal-casual, técnico-simple, serio-humor)
- Card del avatar con campos editables (rol, industria, problemas, objetivos)
- Biblioteca de hashtags con categorías y drag para reordenar
- Sección de ejemplos de tono aplicado (antes/después)
- Diseño tipo settings/configuración organizado en tabs
