# Content Sources

## Overview
Hub de referencias e inspiración para crear contenido. Permite pegar texto de referencia, links de YouTube u otras fuentes, extraer las ideas clave, y transformarlas en posts de LinkedIn con el tono de marca. Funciona como el punto de entrada del flujo creativo: de la referencia al post publicable.

## User Flows
- Pegar un link de YouTube para extraer ideas clave del video
- Pegar texto de referencia (artículo, tweet, nota) como inspiración
- Ver las ideas clave extraídas organizadas como bullet points
- Seleccionar ideas y generar un post de LinkedIn basado en ellas
- Guardar fuentes de referencia en una biblioteca personal
- Ver historial de fuentes usadas previamente

## UI Requirements
- Input principal grande para pegar URL o texto (autodetecta si es link o texto)
- Preview de YouTube embebido cuando se detecta un link de YT (thumbnail + título + duración)
- Panel de "Ideas extraídas" con bullets editables
- Botón "Crear post" que toma las ideas seleccionadas y abre el Content Creator
- Biblioteca/historial de fuentes guardadas como cards
- Tags para categorizar fuentes (video, artículo, tweet, nota personal)
- Diseño tipo inbox/reader, limpio con foco en el contenido
