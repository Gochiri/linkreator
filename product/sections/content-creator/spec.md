# Content Creator

## Overview
Panel principal de LinKreator para generar posts de LinkedIn con tono de marca (estilo Dan Koe / Justin Welsh). Incluye editor de texto con formato, selector de tipo de contenido (tips, storytelling, frameworks, opinión), generación de hashtags optimizados, y vista previa en formato LinkedIn real.

## User Flows
- Seleccionar tipo de contenido (tip, storytelling, framework, opinión, caso de estudio)
- Escribir o generar un post con el tono de marca configurado
- Editar y ajustar el texto generado
- Agregar hashtags sugeridos automáticamente o personalizados
- Previsualizar el post como se vería en LinkedIn
- Copiar el post completo con hashtags para publicar

## UI Requirements
- Editor de texto con contador de caracteres (límite LinkedIn: 3000)
- Selector de formato/tipo de contenido con iconos descriptivos
- Panel lateral de hashtags sugeridos con opción de agregar/quitar
- Vista previa tipo LinkedIn card con avatar, nombre y contenido
- Botón de copiar al portapapeles
- Indicador de tono de marca aplicado
- Diseño limpio, minimalista, con tipografía prominente
