# Canvas

## Overview
Workspace de contenido y knowledge base personal para organizar todo el material que alimenta la creación de contenido en LinkedIn. Inspirado en Eden AI, permite guardar notas en markdown, links de referencia, archivos y recursos organizados en carpetas. Es el centro de conocimiento del creador: donde vive toda la investigación, inspiración y material antes de convertirse en posts.

## User Flows
- Crear una nueva nota en markdown con título y contenido libre
- Guardar un link con título, descripción y tags
- Crear carpetas para organizar notas y links por tema
- Buscar en todo el workspace por texto, título o tags
- Ver notas recientes en la vista principal
- Mover notas y links entre carpetas
- Marcar elementos como favoritos para acceso rápido
- Vista de todos los links guardados con preview

## UI Requirements
- Sidebar izquierdo con árbol de carpetas y secciones (All Notes, Links, Favorites)
- Vista principal tipo grid de cards para notas y links
- Barra de búsqueda prominente en la parte superior
- Botón "+" para crear nuevo elemento (nota, link, carpeta)
- Cards de nota muestran: título, preview del contenido, fecha, carpeta
- Cards de link muestran: título, URL truncada, favicon/preview, tags
- Panel de detalle al hacer click en una nota (editor markdown inline)
- Filtros por tipo (notas, links) y por carpeta
- Diseño limpio tipo workspace, fondo neutro con cards elevadas
- Soporte para tags/etiquetas con colores
